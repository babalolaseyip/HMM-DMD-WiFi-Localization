Overview

This project implements the research paper "WiFi Fingerprinting Indoor Localization Based on Dynamic Mode Decomposition Feature Selection with Hidden Markov Model" by Babalola & Balyan (2021). It provides a complete, production-ready implementation for indoor positioning using WiFi RSSI measurements.

Key Features

üî¨ DMD Feature Extraction

Advanced Dynamic Mode Decomposition for extracting meaningful features from WiFi RSSI time-series data, implementing equations 11-22 from the paper.

üîÑ HMM Modeling

Gaussian Mixture Hidden Markov Models for sequential data modeling and robust location prediction across multiple rooms.

üìä Comprehensive Evaluation

Multiple metrics including mean error, CDF analysis, and accuracy within distance thresholds (1m, 2m, 4m, 6m).

üß™ Synthetic Data

Built-in synthetic data generation for testing and development without requiring the CRAWDAD dataset.

‚úÖ Production Ready

Complete test suite, CI/CD pipeline, Docker support, and proper package structure for deployment.

üìà Visualizations

Automated generation of location plots, error distributions, and cumulative distribution functions.

Quick Start

Installation

# Clone the repository git clone https://github.com/babalolaseyip/HMM-DMD-WiFi-Localization.git cd HMM-DMD-WiFi-Localization # Install dependencies pip install -r requirements/base.txt # Install package in development mode pip install -e .

Basic Usage

# Train with synthetic data python scripts/train_model.py --use-synthetic --output-dir results # Run tests pytest -v # Train with custom config python scripts/train_model.py --config config/default.yaml

Python API

from models.hmm_dmd import HMMDMDLocalizer from src.data.data_loader import CrawdadDataLoader # Load data loader = CrawdadDataLoader() X_train, y_train, rooms = loader.create_synthetic_data(n_samples=1000) # Train model model = HMMDMDLocalizer(dmd_rank=7, n_components=14) model.fit(X_train, y_train, rooms) # Predict predictions = model.predict(X_test)
